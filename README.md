# MyGo!!!!!! 风格个人博客 🥁

一个充满青春乐队气息的个人博客网站，灵感来自 MyGo!!!!!! 的温暖与活力。

## 文件结构

```
mygo-blog/
├── index.html   # 主页面
├── style.css    # 样式表
└── script.js    # 交互脚本
```

## 特色功能

- 🎵 漂浮音符背景动画
- 🥁 CSS 绘制的鼓手 + 鼓组插画
- 💫 滚动渐入动画
- 🏷️ 可交互的标签云
- 🎶 鼠标跟随音符效果
- 🎉 鼓图标彩蛋（点击试试！）

## 使用方法

直接在浏览器中打开 `index.html` 即可预览。
