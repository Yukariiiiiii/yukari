React 19 新特性详解

2026-06-28 3.1k 阅读 10 分钟
React Server Components Actions

React 19 是近年来最大的一次更新，带来了 Actions、Server Components 正式化、use() Hook、自动资源预加载等一系列重磅特性。本文逐一解析，帮你快速上手。

Actions：告别 useState + useEffect 地狱

Actions 是 React 19 最核心的新概念。它让表单提交、数据变更变得极其简洁：

React 19
// 旧方式：useState + onSubmit + useEffect
function OldForm() {
    const [name, setName] = useState('');
    const [loading, setLoading] = useState(false);
    const [error, setError] = useState(null);

    async function handleSubmit(e) {
        e.preventDefault();
        setLoading(true);
        try { await saveName(name); }
        catch (err) { setError(err); }
        finally { setLoading(false); }
    }
    // ... 大量模板代码
}

// ✅ React 19：useActionState
import { useActionState } from 'react';

async function updateNameAction(prevState, formData) {
    const name = formData.get('name');
    const error = await validateName(name);
    if (error) return { error };
    await saveName(name);
    return { success: true };
}

function NameForm() {
    const [state, action, isPending] = useActionState(updateNameAction, {});
    return (
        <form action={action}>
            <input name="name" />
            <button disabled={isPending}>{isPending ? '保存中...' : '保存'}</button>
            {state.error && <p>{state.error}</p>}
        </form>
    );
}
核心优势：Action 自动处理 pending 状态、错误捕获、乐观更新，代码量减少 60% 以上。

use() Hook：读取 Promise 和 Context

新 use() Hook 可以在渲染中直接"等待"一个 Promise：

React 19
import { use } from 'react';

function Comments({ commentsPromise }) {
    // 直接读 Promise，自动 Suspense
    const comments = use(commentsPromise);
    return comments.map(c => <p key={c.id}>{c.text}</p>);
}

// 父组件
<Suspense fallback={<Skeleton />}>
    <Comments commentsPromise={fetchComments()} />
</Suspense>

Server Components 正式落地

RSC 让组件可以在服务端渲染，零 JS 发送到客户端：

React Server Component
// app/page.js (服务端)
import { db } from '@/lib/db';
import { ClientCounter } from './client-counter';

export default async function Page() {
    const posts = await db.post.findMany();
    return (
        <div>
            <h1>最新文章</h1>
            {/* 服务端直接查数据库，零 API 请求 */}
            {posts.map(p => <PostCard key={p.id} post={p} />)}
            {/* 需要交互的部分用 'use client' */}
            <ClientCounter />
        </div>
    );
}

自动资源预加载

React 19 自动处理 <link rel="preload">，无需手动管理：

React 19
// React 自动将 link 提升到 head，并在需要时预加载
function App() {
    return (
        <>
            <link rel="stylesheet" href="/styles.css" />
            <link rel="preload" as="image" href="/hero.png" />
            <Main />
        </>
    );
}

其他值得关注的改动

  • useOptimistic：一行代码实现乐观更新
  • useFormStatus：在表单内获取提交状态
  • ref 作为普通 prop：不再需要 forwardRef
  • context 作为 provider：<Context> 直接当 Provider 用
  • 支持自定义元素：更好的 Web Components 互操作
React 19 - 简化 ref
// ❌ React 18：需要 forwardRef
const Input = forwardRef((props, ref) => <input ref={ref} {...props} />);

// ✅ React 19：ref 就是普通 prop
function Input({ ref, ...props }) {
    return <input ref={ref} {...props} />;
}
