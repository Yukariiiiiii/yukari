深入解析 CSS 毛玻璃效果

2026-07-18 1.2k 阅读 8 分钟
CSS backdrop-filter UI设计

毛玻璃效果（Glassmorphism）是近年来最流行的 UI 设计趋势之一。从 macOS 的菜单栏到 Windows 的 Fluent Design，再到各种现代 Web 应用，模糊半透明的外观给人一种高级、轻盈的视觉感受。本文将深入解析其原理与实战技巧。

什么是毛玻璃效果？

毛玻璃效果本质上是对背景内容进行模糊处理，同时在表面叠加一层半透明的颜色，营造出"透过磨砂玻璃看东西"的视觉效果。它通常包含三个核心要素：

  • 半透明背景：让底层内容隐约可见
  • 背景模糊：使用 backdrop-filter: blur() 实现
  • 细边框：1px 的半透明白边增强玻璃质感

核心 CSS 属性

实现毛玻璃效果只需要一行关键 CSS：

CSS
.glass {
    background: rgba(255, 255, 255, 0.1);
    backdrop-filter: blur(16px) saturate(180%);
    -webkit-backdrop-filter: blur(16px) saturate(180%);
    border: 1px solid rgba(255, 255, 255, 0.15);
    border-radius: 16px;
}
小技巧：saturate(180%) 可以让透过玻璃的颜色更鲜艳，避免模糊后整体发灰。

进阶技巧

1. 多层叠加

通过多层半透明叠加，可以创造出更丰富的深度感：

CSS
.glass-deep {
    background: rgba(255, 255, 255, 0.05);
    backdrop-filter: blur(24px) saturate(200%);
    box-shadow:
        inset 0 1px 0 rgba(255,255,255,0.1),
        0 8px 32px rgba(0, 0, 0, 0.3);
}

2. 动态光晕配合

毛玻璃 + 背景光晕是绝配。在元素背后放置模糊的渐变色块，透过玻璃后会形成非常梦幻的效果（本博客就是这么做的）。

3. 性能优化

  • 避免在滚动元素上使用大面积的 backdrop-filter
  • 优先使用 transform 和 opacity 做动画
  • 必要时用 will-change: backdrop-filter 提示浏览器
注意：在 Firefox 老版本中 backdrop-filter 默认关闭，可通过 layout.css.backdrop-filter.enabled 开启。生产环境建议提供不依赖模糊的降级方案。

完整实战示例

下面是一个完整的毛玻璃卡片组件：

HTML + CSS
<div class="glass-card">
    <h3>毛玻璃卡片</h3>
    <p>透过我可以隐约看到背后的光晕和色彩流动。</p>
</div>

.glass-card {
    padding: 32px;
    background: rgba(255, 255, 255, 0.08);
    backdrop-filter: blur(20px) saturate(180%);
    border: 1px solid rgba(255, 255, 255, 0.12);
    border-radius: 20px;
    box-shadow: 0 8px 32px rgba(0, 0, 0, 0.2);
    transition: transform 0.3s, box-shadow 0.3s;
}
.glass-card:hover {
    transform: translateY(-4px);
    box-shadow: 0 16px 48px rgba(0, 0, 0, 0.3);
}

总结

毛玻璃效果的核心就是 backdrop-filter: blur()，配合半透明背景和细边框就能实现。关键在于：

  • 背景要有内容/色彩才能让模糊效果显现
  • 饱和度提升让透过玻璃的颜色更鲜活
  • 适当的阴影和边框增强立体感和质感
  • 注意性能，大面积使用要谨慎
